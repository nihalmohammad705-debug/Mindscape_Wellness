# MindScape Wellness - Mental Wellness Tracking System

A comprehensive full-stack application for tracking mental wellness, nutrition, and physical activity.

## Features
- Mood tracking and analytics
- Nutrition and calorie tracking
- Activity monitoring
- Personalized recommendations
- Secure user authentication

## Tech Stack
- Frontend: React.js
- Backend: Node.js/Express
- Database: MySQL
- Charts: Chart.js/Recharts

## Quick Start
1. Setup database (see database/schema.sql)
2. Start backend: `cd server && npm start`
3. Start frontend: `cd client && npm start`
